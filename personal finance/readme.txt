# 💬 Personal Finance Chatbot — Hackathon Demo

## 🚀 Overview
An AI-driven personal finance chatbot that:
- Generates **budget summaries**
- Provides **spending insights**
- Adapts advice for **students & professionals**
- Supports **NLP via Watson / HuggingFace**
- Ready for **Granite LLM integration**

---

## 🛠️ Setup

```bash
git clone <repo-url>
cd personal-finance-chatbot
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
